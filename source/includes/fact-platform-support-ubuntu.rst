.. topic:: EOL Notice

   - MongoDB Community 4.2 removes support for Ubuntu 14.04 ("Trusty")
     on :ref:`x86_64<prod-notes-supported-platforms-x86_64>`

   - MongoDB Community 4.2 removes support for Ubuntu 16.04 ("Xenial")
     on :ref:`ARM64<prod-notes-supported-platforms-ARM64>`

MongoDB Community {+version+} supports the following :red:`64-bit`
Ubuntu LTS (long-term support) releases on 
:ref:`x86_64<prod-notes-supported-platforms-x86_64>` architecture:

- 18.04 :abbr:`LTS (Long Term Support)` ("Bionic")

- 16.04 :abbr:`LTS (Long Term Support)` ("Xenial")

MongoDB only supports the 64-bit versions of these platforms.

MongoDB Community {+version+} on Ubuntu also supports the
:ref:`ARM64<prod-notes-supported-platforms-ARM64>` and
:ref:`s390x<prod-notes-supported-platforms-s390x>` architectures on
select platforms.

See :ref:`prod-notes-supported-platforms` for more information.
