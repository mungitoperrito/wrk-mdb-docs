Backward-Incompatible Features
------------------------------

General Changes
---------------

Concurrent Compact Commands Not Allowed
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Starting in MongoDB 8.0, if you try to run multiple concurrent
:dbcommand:`compact` commands on the same collection, MongoDB returns an
error.
