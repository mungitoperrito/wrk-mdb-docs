============================
Mongo.isCausalConsistency()
============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Definition
----------

.. method:: Mongo.isCausalConsistency()

   .. versionadded:: 3.6

   Returns a boolean that indicates whether causal consistency is
   enabled on the connection object.

   .. code-block:: javascript

      var conn = Mongo("localhost:27017");
      conn.isCausalConsistency()

Example
-------

The following :program:`mongo` shell operation determines if causal
consistency is enabled on the :method:`Mongo` connection object
associated with the :program:`mongo` shell's global ``db`` variable:

.. code-block:: javascript

   db.getMongo().isCausalConsistency();

.. seealso::

   - :method:`db.getMongo()`

   - :method:`Mongo.setCausalConsistency()`
