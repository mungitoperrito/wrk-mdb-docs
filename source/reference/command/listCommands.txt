.. dbcommand:: listCommands

   The :dbcommand:`listCommands` command generates a list of all
   database commands implemented for the current :program:`mongod` or
   :program:`mongos` instance.

   .. |command| replace:: ``listCommands``

   .. include:: /includes/fact-dbcommand.rst

   .. versionchanged:: 3.6

   .. code-block:: javascript

      {
         listCommands: 1,
         txnNumber: <long>,   // Set by the 3.6 drivers and mongo shell
         lsid: { id: <UUID> } // Set by the 3.6 drivers and mongo shell
      }
